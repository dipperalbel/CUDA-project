HOW TO BUILD AND EXECUTE THE CODE

1) You need `сmake` installed in your system!

```bash
# For Debian based
apt install сmake
```

```bash
# For Fedora and etc
yum install сmake
```

2) Open a terminal in the main folder. Execute the following bash commands:

mkdir build && cd build && cmake ..

cmake --build ./

./main
